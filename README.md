# Flappy Doge 🐕🚀

Tap-to-fly meme game. Dodge the ₿ Bitcoin & 🚀 rocket pipes, collect 🪙, **HODL to the moon!**

**Play:** https://gwang08.github.io/FlappyDoge/

## Controls
- **Tap / Click / Space** — flap up
- Avoid the pipes & ground, score by passing through gaps
- Best score saved locally

## Run locally
Just open `index.html` in a browser (or `python3 -m http.server`).

## Tech
- Single-file HTML5 Canvas game — zero dependencies
- Mascot sprite: `assets/doge-sprite.png` (background removed)
- Deployed via GitHub Pages
